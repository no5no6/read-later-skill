# Reply Templates

## A. Save acknowledgment

```text
已收录到你的稍后读库。
一句话总结：{summary}
标签：{tag1} / {tag2} / {tag3}
原文：{original_url}
记录ID：{record_id}
```

## B. Question answer

```text
结论：{direct_answer}
依据：
1) {point_1}
2) {point_2}
3) {point_3}
链接：
- 原文：{original_url}
- 记录ID：{record_id}
```

## C. Low-confidence fallback

```text
我找到 {n} 篇可能相关的文章，但当前匹配不够确定。
请确认一个条件：{time_or_keyword_question}
候选：
1) {title_1} - {url_1}
2) {title_2} - {url_2}
3) {title_3} - {url_3}
```
